# Copy conventions

- Bahasa Indonesia aktif, netral, singkat. Nama produk `lembar` (lowercase). Tidak “LembarAI”.
- Sintaks: pakai “buat draft”, “tinjau”, “finalkan”, “siap cetak”. Bukan “AI menghasilkan”.
- Tanggal `id-ID`, zona waktu dari workspace/user setting; tidak hardcode tahun pelajaran.
- Tidak membuat testimonial/statistik/klaim pelanggan tanpa bukti.
- Tidak menyebut nama provider/model AI di UI pengguna.
- Tidak terdengar AI: tidak ada “Hmm”, “Mari kita coba lagi”, emoji untuk menutupi ketidakjelasan.
- `lembar` terasa tenang dan teliti. Bukan karakter AI. Bukan dashboard korporat.
