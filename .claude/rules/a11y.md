# Accessibility

- Standar: WCAG 2.2 AA intent.
- Field: `<label>` di atas `<input>`. `<input aria-describedby>` ke `FieldError`/`help`.
- Skip-link “Lewati ke formulir” di layout `(auth)`. Skip-link “Lewati ke konten utama” di shell app.
- Fokus otomatis pindah ke field invalid pertama.
- Submit state: `aria-busy="true"` di form + `aria-live="polite"` di `FormStatus`.
- 320 CSS px + 200% zoom tanpa overflow horizontal.
- Touch target ≥44×44 pada workflow mobile.
- Reduced motion: transisi non-esensial di-disable saat `prefers-reduced-motion: reduce`.
- Status tidak dengan warna saja: ada label dan/atau ikon.
